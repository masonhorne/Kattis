## Bash Functions
 
#### Functions that can be added to .bash_profile to compile and execute solutions in directory structure.

> To execute scripts run...
>>build \<project-name>

```
 function compile() {
     if [ -f src/$1.cpp ]; then
         c++ -std=c++11 src/$1.cpp -o exec/$1
     else
         echo "Source file doesn't exist: ${1}"
     fi
 }

 function run() {
     if [ -f exec/$1 ]; then
         ./exec/$1  
     else
         echo "Executable file doesn't exist: ${1}"
     fi  
 }

 function build() {
     compile $1
     echo "Please paste example input:"
     run $1
 }
```
